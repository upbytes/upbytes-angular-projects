Getting Started

This page will help you to install Angular Material Calendar library in your project by using angular-cli and npm. You will be able to add a prebuilt theme to your project by placing CSS files in your angular.json.

Install

angular-cli

Run the below command to add this library to your project. 

    
        ng add  @upbytes.in/angular-material-calendar@latest

After running the above command you will be prompt to add prebuild CSS theme to your project. Please choose the same CSS theme which you have selected while installing Angular Material.

Npm

Run the below command to add this library to your project. 

    
        npm install  @upbytes.in/angular-material-calendar

If you choose to install the library by using NPM then you should add a prebuilt CSS theme in your angular.json manually. 

    
        "styles": [
        "projects/upbytes-angular-calendar/src/styles.scss",
        "./node_modules/@upbytes.in/angular-material-calendar/theme/prebuilt/indigo-pink.theme.css"
      ],

If you are working with a custom theme then you can leave a prebuilt theme and check to theme section for more information.

Add AngularMaterialCalendarModule in your module.ts 
    
        import { AngularMaterialCalendarModule } from '@upbytes.in/angular-material-calendar';
        imports: [
         ....MaterialModule,
        StoreModule.forRoot({})
        AngularMaterialCalendarModule
        ]
Add in your component html 
     
        <upbytes-angular-material-calendar [dataSource]='dataSource'>
        </upbytes-angular-material-calendar>

A Datasource could be an array, observable or material Datasource. You can remove [dataSource] if you don't have any event to display.

Peer Dependencies:

Angular Material Calendar library will support Angular12+ version. Apart from angular core, common, cdk, and material it depends upon below other dependencies. 

    
    "peerDependencies": {
        "moment": "^2.29.1",
        "rxjs": "~7.4.0",
        "@angular/material-luxon-adapter": "^13.0.1",
        "@types/luxon": "^2.0.5",
        "@ngrx/store": "^13.0.0-rc.0"
      }

Event

An Event object should contain the below fields.

    
        export interface Event 
        {
            start: number | string | Moment | DateTime | Date,
            end: number | string | Moment | DateTime | Date,
            color: string,
            title?: string,
            description?: string
        }

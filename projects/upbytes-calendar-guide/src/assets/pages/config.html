Angular Material Calendar Config

Anguar Material Calendar provide few option to config Calendar view and format.

Description

view: (day | week | month) view property is used when you need to initialize calendar with day view or week view.

format: format property should be used when your string start/ end date are not be parse by Calendar Api.

Angular Material Calendar use DateTime.fromISO() of Luxon api to parse app string date-time data.

In case, you are facing challanges to parse your input dates then you can use format property.

    
        import { AngularMaterialCalendarModule } from '@upbytes.in/angular-material-calendar';
        imports: [
         ....MaterialModule,
        StoreModule.forRoot({})
        AngularMaterialCalendarModule.forRoot({view: 'month', format: 'YOUR_FORMAT'})
        ]

Angular Material Calendar Config

Angular Material Calendar provides a few options to config Calendar view and format.

Description

view: (day | week | month) View property is helpful to initialize the calendar with day, week, or month view.

format: format property is helpful when your string start or end Calendar is not able to parse start or end date.

Angular Material Calendar uses DateTime.fromISO() of Luxon API to parse app string date-time data.

In case, you are facing challenges in parsing your input dates then you can use the format property.

    
        import { AngularMaterialCalendarModule } from '@upbytes.in/angular-material-calendar';
        imports: [
         ....MaterialModule,
        StoreModule.forRoot({})
        AngularMaterialCalendarModule.forRoot({view: 'month', format: 'YOUR_FORMAT'})
        ]

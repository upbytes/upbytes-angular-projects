Theming & Typography in Angular Material Calendar

Angular Material Calendar built on angular material components and follow theming and typography of same.

Pre-built Themes

Angular Material Calendar includes all four pre-built themes of angular material.

To use pre-built theme, you need to add any of them in styles array of angular.json file.

Theme Light or dark? Palettes (primary, accent, warn) Path
deeppurple-amber.css Light deep-purple, amber, red ./node_modules/@upbytes.in/angular-material-calendar/theme/prebuilt/deeppurple-amber.theme.css
indigo-pink.css Light indigo, pink, red ./node_modules/@upbytes.in/angular-material-calendar/theme/prebuilt/indigo-pink.theme.css
pink-bluegrey.css Dark pink, bluegrey, red ./node_modules/@upbytes.in/angular-material-calendar/theme/prebuilt/pink-bluegrey.theme.css
purple-green.css Dark purple, green, red ./node_modules/@upbytes.in/angular-material-calendar/theme/prebuilt/purple-green.theme.css

Custom Theme

Angular Material Calendar support custom themes feature as well. You can use any color combination according to your requirement.

Follow below steps to create custom theme in Angular Material Calendar.

Step 1:

Import angular-material-calendar.theme in your theme.scss file

    
        @import "../../../node_modules/@upbytes.in/angular-material-calendar/theme/angular-material-calendar.theme";

Step 2:

Define your application theme

    
        @import "../../../node_modules/@upbytes.in/angular-material-calendar/theme/angular-material-calendar.theme";
        
        @include mat.core();
        
        $my-application-dark-primary: mat.define-palette(mat.$pink-palette);
        $my-application-dark-accent: mat.define-palette(mat.$blue-gray-palette);
        $my-application-dark-warn: mat.define-palette(mat.$red-palette);

        $my-application-dark-theme: mat.define-dark-theme(
            (
                color: (
                    primary: $angular-material-calendar-demo-dark-primary,
                    accent: $angular-material-calendar-demo-dark-accent,
                    warn: $angular-material-calendar-demo-warn
                )
            )
        );

Step 3:

Apply your theme on angular material all components and Angular Material Calendar.

    
        @import "../../../node_modules/@upbytes.in/angular-material-calendar/theme/angular-material-calendar.theme";
        
        @include mat.core();
        
        $my-application-dark-primary: mat.define-palette(mat.$pink-palette);
        $my-application-dark-accent: mat.define-palette(mat.$blue-gray-palette);
        $my-application-dark-warn: mat.define-palette(mat.$red-palette);

        $my-application-dark-theme: mat.define-dark-theme(
            (
                color: (
                    primary: $angular-material-calendar-demo-dark-primary,
                    accent: $angular-material-calendar-demo-dark-accent,
                    warn: $angular-material-calendar-demo-warn
                )
            )
        );

        @include mat.all-component-themes($my-application-dark-theme);
        @include angular-material-calendar($my-application-dark-theme);

You can also define light theme of your application by using define-light-theme.

You can test multiple themes by using Theme-Picker button on top-right conrner of this page.

Typography Config

Angular Material Calendar uses same typography as other angular material component in your application.

You can change or use custom fonts by following below steps.

Step 1:

Import your font in index.html file.

Below example import Robot font. 
    
        <head>
            <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet">
            <link rel="preconnect" href="https://fonts.gstatic.com">
            <link href="https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;500&display=swap" rel="stylesheet">
        </head>

Define Typepography in your theme.scss file.

    
        @import "../../../node_modules/@upbytes.in/angular-material-calendar/theme/angular-material-calendar.theme";
        
        @include mat.core();
        
        $my-application-typography: mat.define-typography-config(
            $font-family: 'Roboto'
        );

        $my-application-dark-primary: mat.define-palette(mat.$pink-palette);
        $my-application-dark-accent: mat.define-palette(mat.$blue-gray-palette);
        $my-application-dark-warn: mat.define-palette(mat.$red-palette);

        $my-application-dark-theme: mat.define-dark-theme(
            (
                color: (
                    primary: $angular-material-calendar-demo-dark-primary,
                    accent: $angular-material-calendar-demo-dark-accent,
                    warn: $angular-material-calendar-demo-warn
                )
            )
        );
        @include mat.all-component-themes($my-application-dark-theme);
        @include angular-material-calendar($my-application-dark-theme);

Step 3:

Add typography in your define theme.

    
        @import "../../../node_modules/@upbytes.in/angular-material-calendar/theme/angular-material-calendar.theme";
        
        @include mat.core();
        
        $my-application-font: mat.define-typography-config(
            $font-family: 'Roboto'
        );

        $my-application-dark-primary: mat.define-palette(mat.$pink-palette);
        $my-application-dark-accent: mat.define-palette(mat.$blue-gray-palette);
        $my-application-dark-warn: mat.define-palette(mat.$red-palette);

        $my-application-dark-theme: mat.define-dark-theme(
            (
                color: (
                    primary: $angular-material-calendar-demo-dark-primary,
                    accent: $angular-material-calendar-demo-dark-accent,
                    warn: $angular-material-calendar-demo-warn
                ),
                typography: $my-application-font
            )
        );
        
        @include mat.all-component-themes($my-application-dark-theme);
        @include angular-material-calendar($my-application-dark-theme);

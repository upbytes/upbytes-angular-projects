Introduction

Angular Material Calendar library is developed to display events in a full-page calendar. This library will help you to develop a calendar with day/week/month view with events data and able to display on desktop and mobile platforms.

This calendar library is developed by using Angular 13 and Angular Material 13 components and follows all material design principles.

Theming

It is easy to customize calendar design to match looks and feel of you company by using Material theming. For more detail look into the Theming page.

DateTime

All date-time in the calendar is calculated by numeric values, which help in fast processing and the end-user doesn't need to add any adapter. You can pass the date by using any JS valid format. For more detail look into the Input page.

Views:

Angular Material Calendar library is responsive and able to adjust view by screen size.

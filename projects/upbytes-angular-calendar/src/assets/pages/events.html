Understanding Input Event Type

This page will help to understand input data for Angular Material Calendar.
AngularMaterialCalendarComponent is a generic type class and one can pass any type of input that contains start and end properties.

Event

All examples given below are use event objects. An Event object should contain the below fields.

    
        export interface Event 
        {
            start: number | string | Moment | DateTime | Date,
            end: number | string | Moment | DateTime | Date,
            color: string,
            title?: string,
            description?: string
        }

Description

start: (non-optional) Event start date and time. It can be a number(value in millisecond), string, Moment, Luxon DateTime, or javascript date object.

end: (non-optional) Event end date and time. It can be a number(value in millisecond), string, Moment, Luxon DateTime, or javascript date object.

color: (non-optional) hex value of event background color.

title: (optional) string value to display Event title.

description: (optional) string value to display Event description in event full view.
